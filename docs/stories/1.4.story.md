## Story
Como um desenvolvedor do Churchify, eu quero remover todas as dependencias CDN e alinhar @google/genai para versao npm ^1.30.0, para que o bundle seja auto-contido e a geracao de conteudo AI funcione com a versao correta do pacote.

## Objetivo

Remover todas as dependencias via CDN (importmap em index.html), alinhar versoes de pacotes no package.json (especialmente @google/genai de 0.1.1 para ^1.30.0), e garantir que geminiService.ts funciona com a versao npm do pacote.

### Key Technical Context
- **importmap (index.html lines 29-45):** Loads React 19.2, GenAI 1.30, lucide-react, react-router-dom, Firebase from CDN — ALL must be removed
- **Version conflict:** package.json has `@google/genai: ^0.1.1` but CDN loads 1.30.0 — upgrade npm to ^1.30.0
- **API surface change risk:** GenAI 0.1.1 -> 1.30.0 may have breaking API changes — Task 1 maps differences
- **geminiService.ts:** Uses `process.env.API_KEY` (injected by vite.config.ts define) — verify this still works post-CDN removal
- **Tailwind CDN:** `<script src="https://cdn.tailwindcss.com">` on line 7 — should already be gone after Story 1.1 (PostCSS), but verify
- **Google Fonts:** Keep `<link>` tag for fonts — only remove JS CDN refs
- **Blocked by 1.1** — needs clean build pipeline to verify bundle
