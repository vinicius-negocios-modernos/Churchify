## Story
Como um desenvolvedor do Churchify, eu quero migrar a autenticacao de Firebase Auth para Supabase Auth com Google OAuth e tabela de profiles, para que tenhamos autenticacao segura com persistencia real de dados de usuario.

## Objetivo

Migrar autenticacao de Firebase Auth para Supabase Auth com Google OAuth, criar tabela de user profiles com trigger automatico, remover Firebase SDK e dead imports, e configurar variaveis de ambiente Supabase.

## Debitos Resolvidos

- SYS-001: Firebase credentials hardcoded como placeholders (2h)
- SYS-006: Firestore/Storage dead imports (1h)
- DB-002: Firebase config usa placeholders (absorvido)
- DB-003: No firebase.json / projeto nao inicializado (eliminado pela migracao)
- DB-008: No user profile collection/table (6h)
