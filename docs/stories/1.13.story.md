## Story
Como um desenvolvedor do Churchify, eu quero um pipeline de CI/CD automatizado com GitHub Actions, para que cada PR seja validado automaticamente (lint, typecheck, test, build) e nenhum codigo quebrado chegue ao branch principal.

## Objetivo

Configurar pipeline de CI/CD com GitHub Actions para executar automaticamente lint, typecheck, testes e build em cada PR, garantindo que nenhum codigo quebrado chegue ao branch principal.
