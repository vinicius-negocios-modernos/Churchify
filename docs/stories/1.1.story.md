## Story
Como um desenvolvedor do Churchify, eu quero reestruturar o projeto para convencao src/ com ESLint, Prettier e Tailwind via PostCSS, para que tenhamos uma fundacao tecnica solida e um build pipeline limpo para todas as fases seguintes.

### Key Technical Context
- **Current alias:** `@/` maps to project root in both vite.config.ts and tsconfig.json — must change to `src/`
- **Tailwind CDN:** `<script src="https://cdn.tailwindcss.com">` in index.html line 7 — remove and replace with PostCSS build
- **importmap:** Lines 29-45 in index.html load React 19, GenAI 1.30, Firebase from CDN — this story only removes Tailwind CDN; importmap removal is Story 1.4
- **No existing lint/format scripts** — only `dev`, `build`, `preview` in package.json
- **experimentalDecorators: true** in tsconfig.json — unused, remove
- **Header.tsx** is never imported anywhere — dead code, remove
